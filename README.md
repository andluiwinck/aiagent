# AI Agent

This repository contains an AI agent designed to perform various tasks.

## Getting Started

To get started with the AI agent, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/andluiwinck/aiagent.git
   cd aiagent
   ```

2. **Install dependencies:**

   ```bash
   pip install -r requirements.txt
   ```

3. **Configure the agent:**

   Edit the `config.yaml` file to set up your API keys and other configurations.

4. **Run the agent:**

   ```bash
   python main.py
   ```

## Features

- **Task Automation:** The agent can automate various tasks based on your configuration.
- **Extensible:** Easily extend the agent's capabilities by adding new modules.
- **Configurable:** Customize the agent's behavior through a simple configuration file.
